点开程序后得到的是一个**tkinter**制作的交互界面，其中

* 文本：这里你可以输入中文，注意适当的标点符号提升语音的流畅性。
	   现在，发现一次性输入文本的字数应该是500字以内吧
* 角色：下拉式列表应该可以满足大多数需求



然后只需点击确认，就会自动生成*** xxx.mp3***文件，并自动播放
